Feature

  1. 你可以使用最酷的 Markdown 语法，进行快速创作

  2. 你可以给文章配上精美的封面图和在文章任意位置插入图片

  3. 你可以对文章进行标签分组

  4. 你可以自定义菜单，甚至可以创建外部链接菜单

  5. 你可以在 Windows，MacOS 或 Linux 设备上使用此客户端

  6. 你可以使用 𝖦𝗂𝗍𝗁𝗎𝖻 𝖯𝖺𝗀𝖾𝗌 或 Coding Pages 向世界展示，未来将支持更多平台

  7. 你可以进行简单的配置，接入 Gitalk 或 DisqusJS 评论系统

  8. 你可以使用中文简体或英语

  9. 你可以任意使用应用内默认主题或任意第三方主题，强大的主题自定义能力

  10. 你可以自定义源文件夹，利用 OneDrive、百度网盘、iCloud、Dropbox 等进行多设备同步
